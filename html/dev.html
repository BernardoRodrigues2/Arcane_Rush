Objetivo do Jogo:
Sobreviver ao máximo possível desviando dos fantasmas e coletando grimórios mágicos para ganhar pontos.
Conforme a pontuação aumenta, a velocidade dos inimigos cresce, tornando o jogo cada vez mais desafiador.
No modo multiplayer, dois jogadores competem para ver quem alcança a pontuação primeiro.

Tecnologias:

  • HTML5 Canvas
  • JavaScript
  • CSS3

Descrição:
Arcane Rush é um jogo arcade 2D desenvolvido com HTML5 Canvas,
onde o jogador controla um mago que deve sobreviver a ondas de inimigos enquanto coleta itens mágicos.
O jogo conta com sistema de fases, pontuação, efeitos sonoros,
animações e modo multiplayer local para dois jogadores.
